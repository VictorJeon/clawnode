# 오픈클로 활용법 총정리: 비개발자도 할 수 있는 AI 자동화

오픈클로를 설치했는데 막상 어떻게 써야 할지 모르겠다는 분들이 많습니다. 코딩을 모르면 AI 자동화가 어렵다는 인식이 있는데, 오픈클로는 다릅니다. 이 글에서는 비개발자도 당장 시작할 수 있는 오픈클로 활용법을 정리합니다.

## 오픈클로 기본 동작 방식 이해

오픈클로는 크게 세 가지로 이뤄져 있습니다.

**Gateway:** AI 에이전트의 핵심 엔진입니다. 백그라운드에서 항상 돌면서 요청을 처리합니다.

**채널:** AI 에이전트와 대화하는 창구입니다. 텔레그램이 가장 많이 쓰입니다. 웹 인터페이스, 슬랙, 디스코드도 지원됩니다.

**스킬(Skill):** AI 에이전트가 수행할 수 있는 능력입니다. Gmail 스킬을 설치하면 이메일을 읽고 쓸 수 있고, 캘린더 스킬을 설치하면 일정을 관리할 수 있습니다.

이 구조를 이해하면 "무엇을 할 수 있는지"가 명확해집니다.

## 텔레그램으로 AI에게 일 시키기

가장 기본적인 사용 방법입니다. 텔레그램을 열고 설정한 봇에게 메시지를 보내면 됩니다.

### 일상적인 질문과 업무 지시

자연어로 편하게 말하면 됩니다. 명령어 형식이 아니어도 됩니다.

- "오늘 날씨 어때?"
- "이 글 요약해줘" (URL 또는 텍스트 붙여넣기)
- "영어로 번역해줘"
- "내일 오전 10시에 팀 미팅 알림 설정해줘"

### 파일 처리

텔레그램에 PDF, 이미지, 엑셀 파일을 그냥 보내면 됩니다.

- PDF 보내기 → "이거 핵심 내용 3줄로 정리해줘"
- 스크린샷 보내기 → "여기 나온 영어 텍스트 한국어로 바꿔줘"
- 엑셀 보내기 → "이 데이터에서 이상한 값 찾아줘"

### 슬래시 명령어

자주 쓰는 기능은 `/` 명령어로 빠르게 실행할 수 있습니다.

- `/memory` → AI가 기억하고 있는 내용 확인
- `/status` → 현재 실행 중인 작업 확인
- `/help` → 사용 가능한 명령어 목록

## 스킬 설치로 기능 확장하기

오픈클로의 기본 기능 외에 추가 스킬을 설치해서 능력을 확장할 수 있습니다.

```
오픈클로 스킬 설치 명령어:
/install [스킬 이름]
```

### 많이 쓰이는 스킬

**gog (Google Workspace 연동)**
Gmail, 캘린더, Drive, Docs를 AI가 직접 다룹니다. "오늘 일정 알려줘", "이 파일 Drive에 저장해줘" 같은 요청이 가능합니다.

**github**
GitHub 이슈 조회, PR 상태 확인, 코멘트 작성을 자동화합니다. 개발팀이라면 주간 개발 현황 요약에 바로 쓸 수 있습니다.

**weather**
현재 날씨, 주간 예보를 받아볼 수 있습니다. 매일 아침 날씨 리포트를 설정하면 편리합니다.

**summarize**
URL을 보내면 웹 페이지나 유튜브 영상 내용을 요약합니다. 뉴스 기사, 긴 문서 정리에 유용합니다.

## 스케줄러로 자동 리포트 설정하기

매일 같은 시간에 정해진 작업을 시키고 싶다면 스케줄러를 활용합니다.

텔레그램에서 이렇게 말합니다.

> "매일 오전 8시 30분에 오늘 날씨와 내 구글 캘린더 일정 알려줘"

오픈클로가 이 요청을 cron 작업으로 등록합니다. 이후에는 매일 자동으로 실행됩니다. 다시 변경하고 싶으면 "아침 리포트 취소해줘" 또는 "오전 9시로 바꿔줘"라고 하면 됩니다.

## V3 장기기억 활용하기

오픈클로의 차별점 중 하나가 V3 장기기억 시스템입니다. AI가 이전 대화 내용을 기억합니다.

실제로 이런 사용이 가능합니다.

- "우리 회사 주요 고객사 목록 기억해줘" → 이후 문서 작성 시 자동 활용
- "나는 항상 회의록을 이런 형식으로 써" → 이후 회의록 생성 시 형식 유지
- "이 프로젝트 진행 상황 기록해줘" → 나중에 "프로젝트 현황 알려줘"로 조회

기억은 대화를 종료해도 유지됩니다. AI가 점점 나에게 맞춰진다는 느낌이 이 기능에서 나옵니다.

## 비개발자가 처음 시작하는 순서

처음 오픈클로를 접한다면 이 순서를 따르는 걸 권장합니다.

1. **텔레그램 대화부터 시작합니다.** 편하게 질문하고 요청하면서 AI가 어떻게 반응하는지 봅니다.
2. **자주 쓰는 서비스 스킬 설치합니다.** Gmail이나 구글 캘린더를 많이 쓰면 gog 스킬을 먼저 설치합니다.
3. **반복 업무 하나를 자동화합니다.** "매일 뉴스 요약" 같은 단순한 것부터 시작합니다.
4. **장기기억에 업무 컨텍스트를 쌓습니다.** 회사 정보, 선호 형식, 자주 쓰는 내용을 기억시킵니다.
5. **점점 복잡한 워크플로우로 확장합니다.** 익숙해지면 멀티 스텝 자동화도 가능합니다.

코드를 한 줄도 모르더라도 이 순서대로 하면 1주일 안에 실질적인 시간 절약을 경험할 수 있습니다.
