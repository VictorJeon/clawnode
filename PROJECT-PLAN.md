# ClawNode — Project Plan v1.0
> 마지막 업데이트: 2026-03-03

---

## 1. 프로젝트 개요

### 한 줄 요약
**"월 구독료 0원. 내 책상 위에서 평생 일하는, 나만의 AI 노드."**

### What We Sell
오픈소스 AI 에이전트(OpenClaw)가 사전 설치된 **M4 Mac Mini 턴키 하드웨어**를
출장 방문하여 고객 눈앞에서 언박싱 → 설치 → 교육까지 원스톱으로 제공하는 서비스.

### Why Now
- OpenClaw + Mac Mini 조합이 2026년 최대 AI 트렌드로 부상
- 크몽에 설치 대행은 레드오션(6~15만 원, 20+ 리스팅), **하드웨어 턴키는 아무도 안 함**
- 크립토 커뮤니티 4만 명 텔레그램 채널(불개미)과 즉시 유통 가능

## 3. 타겟 고객

### Primary: 크립토 투자자/트레이더
- **특성:** 보안 극도 민감, 고자산, 기술 호기심 ↑, 노드 운영 개념 익숙
- **Pain Point:**
  - 24시간 시장 모니터링 불가 (잠은 자야지)
  - SaaS 구독료 → 내 API 키를 남한테 맡기기 싫음
  - OpenClaw 직접 설치하기엔 너무 복잡
- **구매 트리거:** "내 눈앞에서 뜯어서 설치해주니까 백도어 걱정 없음"

### Secondary: IT 리터러시 있는 사업가/프리랜서
- 크립토 기능 빼고 범용 AI 비서로 확장 가능
- Phase 2 이후 타겟

## 4. 상품 구성

### 📦 ClawNode Standard — 300만 원 (VAT 별도)

#### Hardware (소유권 100% 이전)
- Apple Mac Mini M4 (16GB / 256GB) **신품 미개봉**
- 전원 케이블, 스티커 등 애플 정품 동봉물 전량

#### Software (ClawOS 풀 세팅)
- **OpenClaw** (최신 안정 버전) — AI 에이전트 런타임
- **Tailscale** — 외부 접속 보안 터널 (군사급 WireGuard 암호화)
- **Ollama + Local LLM** — Llama 3, bge-m3 등 로컬 모델 (데이터 외부 유출 0)
- **Docker** — 샌드박스 격리 (프롬프트 인젝션 방어)

#### Crypto Agents (사전 탑재 3종)
| 에이전트 | 기능 | 연동 |
|----------|------|------|
| 🕵️ **Alpha Watcher** | 특정 지갑/코인 온체인 변동 감지 → 즉시 알림 | Etherscan, DexScreener, Telegram |
| 📰 **News Breaker** | 크립토 뉴스·트위터 자동 수집 → 아침/저녁 요약 브리핑 | Twitter/X, RSS, Telegram |
| 📊 **Portfolio Tracker** | 거래소 잔고 조회, 포트폴리오 P&L 리포트 | Binance/Upbit API, Telegram |

#### 옵션 에이전트 (추가금)
| 에이전트 | 기능 | 추가금 |
|----------|------|--------|
| 🐋 **Whale Alert** | 고래 지갑 추적 + 대량 이체 감지 | +30만 원 |
| 🤖 **Trade Bot** | 지정가/조건부 주문 자동 실행 | +50만 원 |
| 🔍 **DeFi Scanner** | 신규 유동성 풀/에어드랍 모니터링 | +30만 원 |

#### Service (인간의 터치)
- **출장 방문 설치** — 서울/경기 (지방 출장비 별도 or 화상 세팅)
  - 고객 눈앞에서 미개봉 언박싱
  - 원클릭 스크립트로 설치 (설치 중 교육 병행)
- **현장 교육 (2시간)** — 명령법, 프롬프트 작성법, 에이전트 커스텀
- **사후 Q&A (30일)** — 전용 텔레그램 1:1 채널

## 8. 현장 설치 프로토콜

### 사전 준비 (Mason)
1. **마법의 USB** 제작
   - `install.sh` — 원클릭 설치 스크립트 (Docker, OpenClaw, Tailscale, Ollama, 에이전트)
   - 오프라인 캐시 — LLM 모델 파일, Docker 이미지 (인터넷 변수 제거)
2. **5G 에그** 지참 (고객 와이파이 느릴 때 대비)
3. **예비 Mac Mini** 확보 (초기 불량 대비 — Phase 2부터)

### 현장 프로토콜 (약 2시간)
| 시간 | 단계 | 내용 |
|------|------|------|
| 0:00 | 언박싱 | 고객 앞에서 미개봉 씰 확인 → 뜯기 → 전원 ON |
| 0:05 | OS 세팅 | 언어, 와이파이, Apple ID (고객 계정) |
| 0:15 | 스크립트 실행 | USB 꽂고 `install.sh` 한 줄 실행 |
| 0:20 | **교육 시작** | 설치 백그라운드 진행 중 → "OpenClaw이 뭔지" 개념 강의 |
| 0:40 | 에이전트 시연 | Alpha Watcher 라이브 데모 → 텔레그램 알림 확인 |
| 1:00 | 커스터마이징 | 고객 관심 코인/지갑 입력 → 에이전트에 반영 |
| 1:20 | 프롬프트 실습 | 고객이 직접 명령 내려보기 |
| 1:40 | Q&A + 마무리 | 질문 응답 + 텔레그램 지원 채널 입장 |
| 1:50 | Tailscale 세팅 | 고객 폰에 Tailscale 설치 → 원격 접속 경로 확보 |
| 2:00 | 핸드오버 | "이제 이 노드는 고객님 겁니다. 30일간 옆에 있겠습니다." |

### 탈출 전략 (에러 발생 시)
> "보안 터널(Tailscale)은 이미 뚫려 있으니, 무거운 최적화 작업은 제가 사무실 복귀 후 원격으로 안전하게 마무리해 드릴게요."
