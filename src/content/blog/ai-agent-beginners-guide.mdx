# AI 에이전트란? 비개발자를 위한 완벽 입문서

"AI 에이전트 써봤어요?" 이 질문에 많은 분들이 "ChatGPT 쓰는 거 아닌가요?"라고 답합니다. 이해할 수 있습니다. 겉으로 보면 둘 다 AI고, 둘 다 질문에 답해줍니다.

하지만 둘은 근본적으로 다릅니다. 이 차이를 모르면 AI를 활용할 수 있는 범위가 좁아집니다.

## 챗봇과 에이전트의 차이: 직원 비유로

챗봇은 **인턴**과 비슷합니다. 질문을 받으면 답을 줍니다. "서울 날씨 어때?"라고 물으면 알려줍니다. 다음 질문을 받으면 또 답합니다. 스스로 뭔가를 시작하지 않습니다. 기억도 짧습니다.

AI 에이전트는 **자율적으로 일하는 직원**에 가깝습니다. 목표를 줍니다. "이번 달 영업 보고서 정리해줘." 에이전트는 스스로 생각합니다. 어떤 파일을 열어야 하는지, 어떤 데이터를 찾아야 하는지, 결과를 어디에 저장할지. 그리고 실행합니다. 중간에 문제가 생기면 방법을 바꿉니다. 끝나면 보고합니다.

| | 챗봇 | AI 에이전트 |
|---|------|-------------|
| 작동 방식 | 질문 → 답변 (1회) | 목표 → 계획 → 실행 → 완료 |
| 도구 사용 | 없음 (텍스트만) | 검색, 파일 열기, 이메일 전송 등 |
| 기억 | 대화 내에서만 | 장기 기억 가능 |
| 자율성 | 없음 | 스스로 판단하고 행동 |
| 예시 | ChatGPT 기본 | 오픈클로, AutoGPT |

## 에이전트는 실제로 무엇을 할 수 있나

가장 쉬운 이해 방법은 실제 사례를 보는 겁니다.

**사례 1: 이메일 정리**
매일 아침 "오늘 이메일 중 회신 필요한 것 알려줘"라고 합니다. 에이전트가 받은 메일함을 열고, 내용을 읽고, 회신이 필요한 것과 아닌 것을 분류해서 목록으로 만들어줍니다. 이 과정에서 도구(이메일 API)를 직접 사용합니다.

**사례 2: 리서치**
"경쟁사 A, B, C의 최신 가격 정책 조사해줘"라고 합니다. 에이전트가 웹을 검색하고, 각 사이트를 열고, 가격 정보를 수집하고, 표로 정리해서 파일로 저장합니다. 사람이 한다면 1~2시간 걸릴 일을 몇 분 안에 합니다.

**사례 3: 스케줄 관리**
"다음 주 출장 일정 잡아줘. 화요일 오전은 비워놔"라고 하면 캘린더를 확인하고, 비행편을 검색하고, 가능한 일정을 제안합니다.

이 세 가지의 공통점이 있습니다. 에이전트가 **도구를 사용하고**, **여러 단계를 스스로 이어가고**, **중간 판단**을 합니다.

## ChatGPT와 뭐가 다른가

ChatGPT도 "GPT-4o with tools"로 사용하면 일부 에이전트 기능이 있습니다. 웹 검색, 코드 실행, 파일 분석 등입니다.

차이는 **어디서 실행되냐**와 **얼마나 자율적이냐**입니다.

ChatGPT는 브라우저 탭에서 실행됩니다. 탭을 닫으면 멈춥니다. 내 컴퓨터 파일에 접근하려면 수동으로 올려야 합니다. 사용자가 계속 화면 앞에 있어야 합니다.

오픈클로 같은 에이전트는 **내 컴퓨터에서 24시간 실행**됩니다. 내 파일 시스템, 이메일, 캘린더에 직접 접근합니다. 제가 자는 동안에도 스케줄된 작업을 처리합니다. 텔레그램으로 지시를 보내면 컴퓨터 앞에 없어도 실행됩니다.

## "자율적"이라는 것의 한계

에이전트가 자율적으로 일한다고 해서 무조건 옳은 판단을 하는 건 아닙니다.

에이전트는 여전히 실수를 합니다. 잘못된 파일을 열거나, 엉뚱한 정보를 찾거나, 모호한 지시를 잘못 해석하기도 합니다. 그래서 처음 세팅할 때 "어떤 도구에 접근할 수 있는지", "어떤 작업은 확인을 먼저 받아야 하는지"를 명확히 정해두는 게 중요합니다.

에이전트의 자율성을 얼마나 허용할지는 신뢰를 쌓아가면서 조정하는 과정입니다. 처음부터 모든 걸 맡기기보다 작은 작업부터 시작하는 것이 현명합니다.

이 주제는 [AI 에이전트의 한계와 올바른 기대값](/blog/ai-agent-limitations)에서 더 자세히 다룹니다.

## 에이전트를 시작하는 가장 쉬운 방법

에이전트를 쓰려면 서버를 직접 구축해야 하나, 코딩을 해야 하나 걱정하시는 분들이 많습니다.

오픈클로는 설치 후 바로 대화로 쓸 수 있게 설계됐습니다. 코딩 없이 텔레그램으로 연결해서 쓸 수 있습니다. [오픈클로 설치 가이드](/blog/openclaw-install-guide)에 전체 과정이 나와 있습니다.

맥미니에 설치하면 집 컴퓨터가 24시간 AI 비서 역할을 합니다. 스마트폰에서 텔레그램으로 지시하면 집 컴퓨터가 실행합니다.
