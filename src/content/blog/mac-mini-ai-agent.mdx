# 맥미니로 나만의 AI 에이전트 만들기: 왜 맥미니인가

AI 에이전트를 24시간 돌리려면 항상 켜져 있는 컴퓨터가 필요합니다. 라즈베리파이, 중고 서버, NAS, 클라우드 VM 등 여러 선택지가 있는데, 왜 많은 사람들이 맥미니를 선택할까요? 직접 테스트한 결과를 정리합니다.

## 맥미니를 선택하는 핵심 이유 3가지

### 1. 전력 효율이 압도적이다

맥미니 M4의 평균 소비 전력은 **10~20W** 수준입니다. 24시간 풀 가동 기준으로 한 달 전기요금은 약 3,000~5,000원입니다.

비교하면 이렇습니다.

| 기기 | 평균 소비전력 | 월 전기요금(24시간) |
|------|-------------|-------------------|
| 맥미니 M4 | 12~18W | 약 4,000원 |
| 인텔 NUC | 25~45W | 약 8,000원 |
| 구형 PC 서버 | 80~150W | 약 25,000원 |
| AWS t3.medium | - | 약 35,000원(약 $26) |

AWS 같은 클라우드 서버와 비교하면 비용이 완전히 다릅니다. 초기 구매 비용이 들지만 1년 이내에 회수됩니다.

### 2. Apple Silicon의 통합 메모리 구조

맥미니 M4의 가장 큰 강점은 CPU와 GPU가 메모리를 공유하는 **UMA(Unified Memory Architecture)** 구조입니다.

일반 PC에서 GPU 연산을 하려면 CPU 메모리에서 GPU 메모리로 데이터를 복사해야 합니다. 맥미니는 이 과정이 없습니다. 로컬 LLM인 Ollama를 실행하면 이 차이가 크게 납니다.

실제로 Ollama로 `llama3.1:8b` 모델을 실행했을 때, 맥미니 M4 16GB 기준으로 토큰당 생성 속도가 약 **40~60 tokens/sec**입니다. 동급 인텔 PC 대비 3~4배 빠른 수치입니다.

### 3. 조용하고 발열이 없다

AI 에이전트 서버는 사무실이나 집 한쪽에 계속 두게 됩니다. 팬 소음이나 발열 문제는 실제 사용성에 직결됩니다.

맥미니 M4는 일반 작업 시 팬이 거의 돌지 않습니다. LLM 추론 같은 고부하 작업에서도 팬 소음은 35dB 이하입니다. 소음이 민감한 사무실 환경에서도 문제없습니다.

## 다른 선택지와 비교

### 라즈베리파이 5

장점: 가격이 저렴합니다(약 10만 원 내외). 소비 전력도 낮습니다.

단점: RAM이 최대 8GB로 제한적입니다. LLM 모델 실행이 느리고, 7B 파라미터 이상 모델은 사실상 사용하기 어렵습니다. 오픈클로 같은 AI 에이전트 프레임워크를 실행하기에 성능이 부족합니다.

### NAS (시놀로지, QNAP)

장점: 스토리지 관리와 함께 사용하기 좋습니다.

단점: CPU 성능이 낮고, x86 기반 NAS는 소비 전력이 높습니다. GPU가 없어서 LLM 추론이 매우 느립니다.

### 클라우드 서버 (AWS, GCP, Azure)

장점: 초기 비용이 없고 언제든 스케일 업이 가능합니다.

단점: 월 비용이 상당합니다. t3.medium 기준 약 $26/월, GPU 인스턴스는 $200 이상입니다. 데이터가 외부 서버에 저장된다는 보안 이슈도 있습니다.

## 맥미니 M4 구성 선택 가이드

오픈클로 AI 에이전트 서버 용도라면 아래 기준으로 선택합니다.

**기본 오픈클로 사용 (클라우드 API 연동):** 맥미니 M4 8GB로 충분합니다. 로컬 LLM 없이 Claude나 GPT API만 사용하면 메모리 요구사항이 낮습니다.

**로컬 LLM 병행 실행:** 16GB 필수입니다. Ollama로 8B~14B 파라미터 모델을 동시에 돌리려면 16GB는 기본입니다.

**멀티 에이전트 + 로컬 LLM:** 24GB 이상을 권장합니다. 여러 에이전트가 동시에 작업하는 환경이라면 메모리가 부족하면 성능 저하가 바로 나타납니다.

## 실제 활용 예시

맥미니에 오픈클로를 설치한 뒤 실제로 아래 작업들을 자동화하고 있습니다.

- 매일 오전 뉴스 요약 → 텔레그램 전송
- 이메일 초안 자동 생성 (Gmail 연동)
- 슬랙 채널 요약 리포트
- 코드 리뷰 자동화 (GitHub 연동)
- 일정 정리 및 알림 (Google Calendar 연동)

이 모든 작업이 맥미니 한 대에서, 월 5,000원 미만의 전기요금으로 돌아갑니다.
