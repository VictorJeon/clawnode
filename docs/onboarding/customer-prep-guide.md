## 1) Telegram 봇 만들기 (모든 티어 필수)

OpenClaw은 Telegram으로 대화합니다. 먼저 내 봇을 하나 만듭니다.

### 1-1. BotFather에서 생성

1. Telegram에서 **@BotFather** 검색
2. `/newbot` 입력
3. 봇 이름 입력 (예: 나의 AI 비서)
4. 봇 아이디 입력 (예: `my_ai_bot`, 반드시 `_bot`으로 끝남)
5. 발급된 **봇 토큰** 복사

```
예시: 1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ
```

> ⚠️ 봇 토큰은 비밀번호입니다. 다른 사람과 공유하지 마세요.

### 1-2. 채팅 시작

1. 만든 봇을 검색해서 대화창 열기
2. `/start` 전송
3. 아무 메시지 1개 더 보내기 (예: 안녕)

> [스크린샷 자리] BotFather 토큰 발급 화면
> [스크린샷 자리] 봇 대화창에서 /start 보낸 화면

## 4) 설치 후 OpenClaw 활용 팁 (실사용 예시)

아래처럼 말하면 바로 써먹을 수 있습니다.

### 자주 쓰는 요청 예시 (10개+)
1. `오늘 일정 정리해줘.`
2. `내일 오전 9시에 물 마시기 알림 설정해줘.`
3. `이 기사 링크 3줄로 요약해줘.`
4. `이 유튜브 영상 핵심만 정리해줘.`
5. `이 PDF 파일에서 중요한 내용만 뽑아줘.`
6. `이 이미지에 뭐가 보이는지 설명해줘.`
7. `이번 주 서울 날씨 요약해줘.`
8. `회의록을 깔끔한 할 일 목록으로 바꿔줘.`
9. `고객에게 보낼 정중한 답장 문구 써줘.`
10. `이 코드 오류 원인 찾아줘.`
11. `초보자도 이해하게 엑셀 함수 설명해줘.`
12. `이 음성 메시지를 텍스트로 바꿔줘.`

### 어떤 기능을 쓸 수 있나요?
- **음성 → 텍스트 변환**: 음성 파일/메시지 받아서 텍스트로 변환
- **파일 분석**: PDF, 이미지, 문서 핵심 요약
- **웹 검색 + 요약**: 최신 정보 찾아서 짧게 정리
- **일정/리마인더 관리**: 할 일, 약속, 알림 정리
- **코딩 도움**: 오류 원인, 코드 설명, 개선 방향 제안

> [스크린샷 자리] 파일 업로드 후 요약받는 화면
> [스크린샷 자리] 알림 설정/일정 정리 대화 화면

## 5) 자주 쓰는 명령어 (운영/점검용)

아래 명령어는 **터미널**에서 입력합니다.

```bash
# 게이트웨이 상태 확인
openclaw gateway status

# 게이트웨이 시작
openclaw gateway start

# 게이트웨이 중지
openclaw gateway stop

# 게이트웨이 재시작 (문제 있을 때 가장 먼저)
openclaw gateway restart

# 도움말 보기
openclaw help
openclaw gateway --help
```

### 로그(오류 내역) 확인
1. 먼저 `openclaw gateway status` 실행
2. 표시되는 상태/오류 메시지 캡처
3. 담당자에게 캡처 전달

### 모델 변경 방법
- 가장 쉬운 방법: 담당자에게 아래처럼 요청
```
현재 모델을 {원하는 모델명}으로 변경하고 싶어요.
비용/성능 기준으로 추천도 부탁드립니다.
```
- 직접 변경이 필요한 경우: `openclaw help` 결과를 담당자와 함께 확인해 진행

### 스킬 추가 방법
- 가장 쉬운 방법: 담당자에게 아래처럼 요청
```
{원하는 작업}을 자동으로 하게 스킬 추가하고 싶어요.
추천 스킬 설치 부탁드립니다.
```
- 직접 진행 시: `openclaw help`로 현재 환경에서 가능한 명령 먼저 확인

> [스크린샷 자리] 터미널에서 status/restart 실행 화면

## 7) FAQ

### Q1. 봇이 갑자기 안 돼요.
가장 먼저 아래 순서로 해주세요.
1) Telegram 봇 대화창에 다시 메시지 보내보기
2) 터미널에서 `openclaw gateway restart` 실행
3) 여전히 안 되면 `openclaw gateway status` 결과 캡처 후 전달

### Q2. 다른 AI로 바꾸고 싶어요.
가능합니다. Claude ↔ ChatGPT ↔ Gemini 등 변경할 수 있습니다.
다만 모델별로 성능/비용이 달라서, 사용 패턴 기준으로 추천받는 게 가장 좋습니다.

### Q3. 비용이 추가로 나오나요?
- 설치비: 결제한 상품 금액 1회
- AI 구독/API: 선택한 서비스 정책에 따라 별도
- DELUXE/PREMIUM 서버: 서버 업체 월 비용 별도

### Q4. 대화 내역은 어디에 저장되나요?
기본적으로 사용 중인 메시지 채널(예: Telegram)과 운영 환경 로그에 남을 수 있습니다.
민감한 개인정보/비밀번호는 채팅에 남기지 않는 것을 권장합니다.

### Q5. 여러 명이 같이 쓸 수 있나요?
가능합니다. 다만 계정/권한/알림이 섞이지 않게 사용자별 운영 규칙을 먼저 정하는 게 좋습니다.
팀 사용이 목적이면 설치 시점에 미리 알려주세요.
