## 봇 사용법

### 기본 대화
Telegram에서 봇에게 메시지를 보내면 됩니다. 예시:

- "안녕" → 인사
- "오늘 서울 날씨 알려줘" → 날씨 검색
- "이 기사 요약해줘 https://..." → URL 요약
- "내일 오전 10시에 회의 알림 줘" → 리마인더
- "영어로 번역해줘: 안녕하세요" → 번역

### 파일 보내기
- 이미지를 보내면 → 분석 & 설명
- PDF를 보내면 → 내용 요약
- 음성 메시지를 보내면 → 텍스트로 변환 후 답변

### 봇 관리 명령어

Telegram에서 직접 입력:

| 명령어 | 설명 |
|--------|------|
| `/status` | 봇 상태 확인 |
| `/reasoning` | 추론 모드 토글 |

터미널에서 (고급):

```bash
openclaw gateway status    # 상태 확인
openclaw gateway restart   # 재시작
```

## 자주 묻는 질문

**Q. 봇이 응답을 안 해요**
→ 터미널에서 `openclaw gateway restart` 실행해보세요. 그래도 안 되면 연락주세요.

**Q. 컴퓨터를 껐다 켜면 봇도 꺼지나요?**
→ 아니요. 자동 시작되도록 설정되어 있습니다. 부팅 후 약 30초 뒤 자동 실행됩니다.

**Q. 다른 AI 모델로 바꾸고 싶어요**
→ 연락주시면 변경해드립니다. 또는 `openclaw onboard`로 직접 변경 가능합니다.

**Q. 대화 내용은 어디에 저장되나요?**
→ 내 컴퓨터 `~/.openclaw/agents/` 폴더에만 저장됩니다. 외부 서버로 전송되지 않습니다.

**Q. 비용이 추가로 나오나요?**
→ AI 모델 구독료(Claude Pro $20/월 등)만 본인 부담입니다. OpenClaw 자체는 무료입니다.

**Q. 여러 명이 같이 쓸 수 있나요?**
→ Telegram 그룹에 봇을 추가하면 가능합니다. 설정이 필요하니 연락주세요.

봇의 말투나 성격을 바꾸고 싶으면:

1. 터미널에서 `nano ~/.openclaw/workspace/SOUL.md` 실행
