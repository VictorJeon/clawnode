## 바이브코딩 강의 요약

### 강의에서 다룬 내용

| 세션 | 주제 | 시간 |
|------|------|------|
| 1 | AI 코딩 도구 소개 (Cursor, Claude Code, Codex) | ~45분 |
| 2 | 실습: 앱 기획 → 코드 생성 → 수정 | ~90분 |
| 3 | 배포 + 유지보수 방법 | ~45분 |

### 핵심 개념

**바이브코딩이란?**
코드를 직접 짜지 않고, AI에게 자연어로 지시해서 앱을 만드는 방법입니다.

**워크플로우:**
```
1. 요구사항을 자연어로 작성
2. AI가 코드 생성
3. 결과 확인 → 수정 지시
4. 반복 → 완성
5. 배포
```

### 사용한 도구

| 도구 | 용도 | 가격 |
|------|------|------|
| **Cursor** | 코드 에디터 + AI | $20/월 (Pro) |
| **Claude Code** | 터미널 기반 AI 코딩 | Claude 구독에 포함 |
| **Codex** | 클라우드 기반 AI 코딩 | ChatGPT 구독에 포함 |
| **Vercel** | 웹앱 배포 | 무료 (Hobby) |
| **GitHub** | 코드 저장소 | 무료 |
